# CARPET Token

The CARPET token is the value token of the protocol. It always increases in value.

It is an uncapped supply token, meaning there can be an infinite amount of CARPET. The Counter (our exchange) has the sole rights to mint CARPET, and it does so, minting on buy and burning on sell.
